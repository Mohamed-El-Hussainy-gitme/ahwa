# Ahwa ☕️
نظام إدارة قهوة بدون كاشير (موبايل أولًا) مبني بـ **Next.js (App Router)** و **Supabase**.

## المميزات الأساسية
- تسجيل دخول:
  - صاحب/معلم
  - موظف (اسم + PIN)
- ورديات (فتح/غلق + تعيين موظفين)
- طلبات ومطبخ/شيشة (حسب الدور)
- تحصيل وفواتير / Billing
- يعمل على الموبايل بشكل أساسي

---

## المتطلبات
- Node.js 18+ (يفضل 20+)
- حساب Supabase + Project جاهز
- npm أو pnpm
