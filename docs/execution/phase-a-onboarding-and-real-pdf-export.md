# Phase A - onboarding guide + real PDF export

## What changed
- The owner onboarding guide now tracks real progress, reloads after returning to the page, shows a progress bar, and can be skipped/hid per cafe.
- Printable pages now include a real `Save PDF` action powered by browser-side canvas capture and local vendor bundles (`html2canvas` + `jsPDF`).
- Existing print pages remain available for paper printing, but they now support direct PDF download as well.

## Scope note
- Support inbox / technical support messages are intentionally not part of this phase.
- The login support form requirements were kept in mind for the next support phase.
