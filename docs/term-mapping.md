# Term mapping

| legacy term | canonical term | note |
| --- | --- | --- |
| table | service session label | no fixed tables in daily runtime |
| table session | service session | session-label-first |
| bill account | delivered billable quantities | no bill-account container |
| deferred account | debtor name ledger | debt is tracked in `deferred_ledger_entries` |
| kitchen ticket | fulfillment event / station queue item | quantity-first |
